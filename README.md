# Blend Mix Optimizer

A Streamlit-based web application for optimizing blend mix calculations.

---

## 🚀 Getting Started

Follow the steps below to run this project locally.

### 1️⃣ Clone the Repository

Open your terminal and run:

```bash
git clone https://github.com/charan-x16/Blend-Mix-Optimizer.git
```

---

### 2️⃣ Open the Project in VS Code

Navigate into the project folder:

```bash
cd Blend-Mix-Optimizer
```

Then open it in VS Code:

```bash
code .
```

---

### 3️⃣ Install Dependencies

Make sure you have **uv** installed.

Run:

```bash
uv sync
```

This will install all required dependencies.

---

### 4️⃣ Run the Application

Start the Streamlit app with:

```bash
streamlit run app.py
```

After running the command, Streamlit will provide a local URL (usually http://localhost:8501).  
Open it in your browser to use the application.

---

## 📌 Requirements

- Python 3.10+
- uv
- Streamlit
